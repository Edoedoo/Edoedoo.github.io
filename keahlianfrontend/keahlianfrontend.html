Frontend

Front end ialah salah satu bagian dari website yang menampilkan tampilan untuk para pengguna. Menggunakan tiga bahasa pemrograman web yaitu HTML,CSS dan JAVASCRIPT .Front end memiliki tanggung jawab dalam menghubungkan situs website ataupun aplikasi dengan para penggunanya. Tugas Front end adalah memindahkan desain yang dibuat UI Designer dalam bentuk yang lebih interaktif dan membuat desain menjadi lebih hidup.

HTML,hypertext markup language

tingkat keahlian

85%

Bahasa markup yang umumnya digunakan untuk menyusun bagian paragraf,heading,maupun link pada halaman web.

CSS,cascading style sheet

tingkat keahlian

80%

Yang digunakan untuk menambahkan gaya/style seperti Font,warna,spasi ataupun animasi yang sudah dibuat strukturnya oleh HTML.

JAVASCRIPT

tingkat keahlian

75%

Sama halnya seperti HTMLdan CSS.Javascript berfungsi untuk membuat halaman yang interaktif.
